# Sherbert
[Sherbert is no longer actively maintained.]
Sherbert (not *sherbet*, stylized with the r) is a simple "search engine" built to make DuckDuckGo's bangs compatible with Brave Search.

Sherbert also makes bangs more efficient than their original implementation in DuckDuckGo.
